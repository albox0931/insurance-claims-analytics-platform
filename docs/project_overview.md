# Project Overview

This project simulates an insurance claims analytics platform.

## Objective
Build a scalable ETL pipeline that converts raw insurance claims data into clean analytics-ready tables.

## Data Sources
- Claims transactions
- Policy information
- Customer information

## Business Users
- Claims operations team
- BI developers
- Business analysts
- Risk and fraud analytics teams

## Main Outcome
Cleaned and modeled data for dashboards and reporting.
